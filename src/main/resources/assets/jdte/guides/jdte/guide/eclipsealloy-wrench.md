# 蚀空合金扳手

<ItemImage id="jdte:eclipsealloy_wrench" scale="2" />

蚀空合金扳手是用于操作 JDTE 和 JDT 机器的多用途工具。

## 使用方法

### 调节方向

右键点击任意 JDTE 或 JDT 机器，将其旋转 90 度。

### 拆下搬运

蹲下 + 右键点击 JDTE 或 JDT 机器，将其整体拆下为方块物品，保留内部物品、能量、流体、升级和配置。

蚀空合金扳手属于通用扳手标签。对 AE2 等支持标准扳手接口的模组方块，普通右键和蹲下右键会交给对应模组处理旋转或快捷拆除，并使用该模组自身的数据保存与权限检查逻辑。

### 调节作用范围

手持蚀空合金扳手，按住**扳手范围修饰键**（默认左 Alt，可在按键绑定中修改），对准 JDTE 或 JDT 中可调节范围的机器滚动鼠标滚轮，可同时增大或减小 X/Y/Z 范围半径。

安装了 FTB Ultimine 时，只有手持蚀空合金扳手、按住扳手范围修饰键、同时按住 FTB Ultimine 连锁按键并存在有效选区时，滚轮调节才会对选区内所有匹配的机器批量生效。未按住连锁按键时，只会调节准星指向的机器。

### 框选作用范围

手持蚀空合金扳手左键方块设置第一个角点，再左键另一个方块设置第二个角点，每次选点都有提示音。框选过程中会使用 JDT 的范围效果预览当前区域，并在准星下方持续显示 X/Y/Z 尺寸。完成两个角点后选择会锁定，不能继续设置新角点；对准选区六个面之一，按住 Ctrl 并滚动滚轮，可以只向内或向外移动所指向的边界面，选区不会缩小到 1 格以下。玩家位于选区内部时滚轮方向会自动反转，与 Create 的区域选择交互一致。

左键可调范围机器即可将框选区域精确写入机器，范围超过当前范围升级允许的半径或偏移上限时不会应用。成功写入后框选仍会保留，可连续应用到多台机器；只有 Shift+左键手动清空后才能开始下一次框选。

创造模式下手持蚀空合金扳手不会破坏方块，避免选取角点时瞬间挖除目标方块。
