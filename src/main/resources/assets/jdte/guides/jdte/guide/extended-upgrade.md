# 扩展升级

<ItemImage id="jdte:extended_upgrade" scale="2" />

扩展升级可以将高级机器直接转换为扩展高级机器，获得 8 个升级槽。

## 使用方法

手持扩展升级，右键点击任意高级机器即可完成转换。转换后：

- 机器变为对应的扩展高级机器
- 保留原有物品和设置
- 扩展升级物品会被消耗

## 支持的机器

| T2 机器 | 转换为 |
|---------|--------|
| 点击器 T2 | 扩展高级点击器 |
| 方块破坏器 T2 | 扩展高级方块破坏器 |
| 方块放置器 T2 | 扩展高级方块放置器 |
| 方块替换器 T2 | 扩展高级方块替换器 |
| 投掷器 T2 | 扩展高级投掷器 |
| 传感器 T2 | 扩展高级传感器 |
| 流体收集器 T2 | 扩展高级流体收集器 |
| 流体放置器 T2 | 扩展高级流体放置器 |
| 高级时间加速器 | 扩展高级时间加速器 |
