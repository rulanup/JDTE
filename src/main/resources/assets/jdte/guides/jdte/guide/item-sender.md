# 物品放置器

物品放置器可以将物品发送到配置范围内的目标容器中。

## 变体

| 机器 | 升级槽 | 每次发送数量 | 需要 FE |
|------|--------|-------------|---------|
| 初级物品放置器 | 4 | 64 | 否 |
| 高级物品放置器 | 4 | 64 | 是 |
| 扩展物品放置器 | 8 | 64 | 是 |

## 功能

- 将内部存储的物品发送到范围内的容器
- 支持红石控制、范围升级、过滤升级
- 安装超频或创造升级并开启自动输入后，可将自动输入侧容器中的物品直接发送到范围目标，默认每次最多 10,000 件，不受内部槽位容量限制
- 速度按钮和降频升级控制执行间隔，批量可在配置中调整
