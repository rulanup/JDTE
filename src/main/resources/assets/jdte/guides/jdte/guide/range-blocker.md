# 范围屏蔽器

<BlockImage id="jdte:range_blocker" scale="2" />

范围屏蔽器复用高级传感器式界面和八个升级槽，通过按区块索引的事件逻辑控制设定范围，不会由机器每 Tick 扫描整个区域。

## 主要模式

- **围困模式**：匹配目标进入范围后会被限制在边界内。目标模式可选择敌对生物、被动生物、所有生物、指定实体类型、非生物实体或所有类型。服务器与客户端在实体自身更新后只纠正越界坐标，并保留未越界方向的速度。玩家和载具关系始终受到保护，命名、驯服和 Boss 生物保护可在配置中调整。非玩家投射物（包括凋零之首、龙火球和大多数模组炮弹）会按拥有者目标类型匹配并在下一步碰撞箱越界前被销毁；非生物/所有类型模式也可直接匹配投射物，玩家发射物仍不受影响。
- **退磁模式**：范围内掉落物会获得通用远程移动屏蔽标记，阻止玩家磁铁吸取，同时允许机器式收集器继续工作。离开范围或机器停机后，由 JDTE 添加的标记会被移除。

围困模式的分类与名单规则和实体抑制器一致，并将过滤槽中的刷怪蛋作为实体类型过滤。退磁模式将普通物品作为精确物品过滤，此时目标模式按钮变灰且不参与匹配，名单按钮仍可使用。空过滤器默认匹配当前分类的全部目标，名单按钮可切换白名单和黑名单。

Simple Magnets 和 Sophisticated 系列通过 `PreventRemoteMovement` / `AllowMachineRemoteMovement` 标记兼容。Mekanism 磁力吸引单元使用默认开启的可选窄兼容入口，在原候选掉落物列表中过滤退磁实体。

投射物围困、无所有者投射物围困和投射物爆炸范围限制均可独立配置且默认开启。被围困投射物在范围内发生爆炸时，范围外方块和实体会从该次爆炸的影响列表中移除。该处理只在投射物 Tick 或实际爆炸时运行，不扫描整个范围。

机器支持范围、过滤、容量和创造升级。空闲时不消耗工作能量；存在匹配目标时，每台机器每 Tick 最多结算一次。围困模式默认消耗 250 FE/工作 Tick；退磁模式默认平均消耗 0.25 FE/工作 Tick，以小数累计方式每 4 个工作 Tick 扣除 1 FE，两项均可独立配置。
