# 粘胶激活器

粘胶激活器用于自动化 JDT 粘胶相关操作，可以自动激活和处理粘胶方块。

## 变体

| 机器 | 升级槽 | 需要 FE |
|------|--------|---------|
| 初级粘胶激活器 | 4 | 否 |
| 高级粘胶激活器 | 4 | 是 |
| 扩展粘胶激活器 | 8 | 是 |

## 功能

- 自动检测并激活范围内的粘胶方块
- 支持红石控制、范围升级、过滤升级
- 高级和扩展版本支持超频、降频和容量升级
