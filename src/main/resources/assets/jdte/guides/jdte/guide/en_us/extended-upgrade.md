# Extended Upgrade

<ItemImage id="jdte:extended_upgrade" scale="2" />

The Extended Upgrade can directly convert advanced machines into extended advanced machines, granting 8 upgrade slots.

## Usage

Hold the Extended Upgrade and right-click any advanced machine to convert. After conversion:

- The machine becomes the corresponding extended advanced machine
- Original items and settings are preserved
- The Extended Upgrade item is consumed

## Supported Machines

| T2 Machine | Converts To |
|------------|-------------|
| Clicker T2 | Extended Clicker |
| Block Breaker T2 | Extended Block Breaker |
| Block Placer T2 | Extended Block Placer |
| Block Swapper T2 | Extended Block Swapper |
| Dropper T2 | Extended Dropper |
| Sensor T2 | Extended Sensor |
| Fluid Collector T2 | Extended Fluid Collector |
| Fluid Placer T2 | Extended Fluid Placer |
| Advanced Time Accelerator | Extended Time Accelerator |
