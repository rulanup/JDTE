# 物品接收器

物品接收器可以从配置范围内的目标容器中接收物品。

## 变体

| 机器 | 升级槽 | 每次接收数量 | 需要 FE |
|------|--------|-------------|---------|
| 初级物品接收器 | 4 | 64 | 否 |
| 高级物品接收器 | 4 | 64 | 是 |
| 扩展物品接收器 | 8 | 64 | 是 |

## 功能

- 从范围内的容器中接收物品到内部存储
- 支持红石控制、范围升级、过滤升级
- 安装超频或创造升级并开启自动输出后，可将范围来源容器中的物品直接发送到自动输出侧容器，默认每次最多 10,000 件，不受内部槽位容量限制
- 速度按钮和降频升级控制执行间隔，批量可在配置中调整
