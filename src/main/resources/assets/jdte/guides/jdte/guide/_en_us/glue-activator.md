# Glue Activator

The Glue Activator automates JDT glue-related operations, automatically activating and processing glue blocks.

## Variants

| Machine | Upgrade Slots | Requires FE |
|---------|--------------|-------------|
| Basic Glue Activator | 4 | No |
| Advanced Glue Activator | 4 | Yes |
| Extended Glue Activator | 8 | Yes |

## Features

- Automatically detects and activates glue blocks within range
- Supports redstone control, Range Upgrade, and Filter Upgrade
- Advanced and Extended versions support Overclock, Underclock, and Capacity Upgrades
