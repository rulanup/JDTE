# 流体接收器

流体接收器可以从配置范围内的目标容器中接收流体。

## 变体

| 机器 | 升级槽 | 每次接收量 | 需要 FE |
|------|--------|-----------|---------|
| 初级流体接收器 | 4 | 20,000 mB | 否 |
| 高级流体接收器 | 4 | 20,000 mB | 是 |
| 扩展流体接收器 | 8 | 20,000 mB | 是 |

## 功能

- 从范围内的流体容器中接收流体到内部储罐
- 支持红石控制、范围升级、过滤升级
- 安装超频或创造升级并开启自动输出后，可将范围来源容器中的流体直接发送到自动输出侧容器，不受内部储罐容量限制
- 速度按钮和降频升级控制执行间隔，普通与超频批量可在配置中调整
