# 生命提取器

生命提取器可以杀死配置范围内的生物，并根据其最大生命值产生生命流体。与生物粉碎机不同，生命提取器不产生掉落物，而是将生物的生命力直接转化为流体。

## 变体

| 机器 | 升级槽 | 能量容量 | 流体容量 |
|------|--------|----------|----------|
| 高级生命提取器 | 4 | 100,000 FE | 16,000 mB |
| 扩展生命提取器 | 8 | 200,000 FE | 16,000 mB |

## 功能

### 生命流体生成

- 杀死配置范围内的生物
- 根据生物最大生命值产生生命流体：每点生命值 100 mB
- 流体存储在内部储罐中，可用桶取出

### 模式

生命提取器支持 3 种模式：

- **敌对**：只攻击敌对生物（默认）
- **友好**：只攻击友好生物
- **全部**：攻击所有生物

安装过滤升级后，将忽略模式设置，改为使用过滤槽配置目标实体。

### 范围

- 基础半径：5 格（X/Y/Z 各方向）
- 可通过 GUI 调整范围
- 安装范围升级可扩大可配置上限
- 能量消耗随范围增大而增加

### 升级效果

| 升级 | 效果 |
|------|------|
| 超频 | 每 tick 处理 2 个生物，但有 10% 流体损耗 |
| 降频 | 流体产量提升至 1.5 倍 |
| 创造 | 每 tick 处理 2 个生物，无损耗，免能耗 |
| 范围 | 扩大可配置范围上限 |
| 过滤 | 启用过滤槽，按实体类型筛选目标 |
| 容量 | 提升 FE 和流体容量 |

## 能量

- 基础能量消耗：300 FE × 范围缩放系数
- 高级生命提取器：100,000 FE 容量
- 扩展生命提取器：200,000 FE 容量
