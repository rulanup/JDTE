# 流体放置器

流体放置器可以将内部流体发送到配置范围内的目标。

## 变体

| 机器 | 升级槽 | 每次发送量 | 需要 FE |
|------|--------|-----------|---------|
| 初级流体放置器 | 4 | 1,000 mB | 否 |
| 高级流体放置器 | 4 | 4,000 mB | 是 |
| 扩展流体放置器 | 8 | 4,000 mB | 是 |

## 功能

- 将内部储罐中的流体发送到范围内的流体容器
- 支持红石控制、范围升级、过滤升级
- 高级和扩展版本支持超频（发送量翻倍）、降频（发送量减半）和容量升级
