# JDT Extras

Just Dire Things 的扩展模组，为 JDT 机器添加升级系统和时间加速器。

## 功能

### 升级卡（7种）

| 升级卡 | 效果 | 每台机器上限 |
|---|---|---|
| 容量升级 | FE 和流体容量翻倍 | 3 |
| 超频升级 | 强制 1 tick 运行，每 tick 执行两次，耗电 3x | 1 |
| 降频升级 | 强制 40 tick 运行，耗电降低 80% | 1 |
| 流体升级 | 流体容量翻倍 | 3 |
| 流体存储升级 | 为 Clicker 添加内部流体储罐 | 1 |
| 发电机升级 | 消耗 2x 燃料，产出 3x FE | 1 |
| 范围升级 | 区域半径上限翻倍 | 2 |

### 时间加速器

- **初级时间加速器** — 仅消耗时间流体，默认 4x 加速，超频后 16x
- **高级时间加速器** — 消耗时间流体 + FE，可调节 1-128x，超频后 256x

### 扩展高级机器（8个升级槽）

- 扩展高级点击器
- 扩展高级方块破坏器
- 扩展高级方块放置器
- 扩展高级方块替换器
- 扩展高级投掷器
- 扩展高级传感器
- 扩展高级流体收集器
- 扩展高级流体放置器

## 安装

1. 安装 [NeoForge](https://neoforged.net/) 1.21.1
2. 安装 [Just Dire Things](https://www.curseforge.com/minecraft/mc-mods/just-dire-things)
3. 将 `jdte-x.x.x.jar` 放入 `mods` 文件夹

## 依赖

- Minecraft 1.21.1
- NeoForge 21.1.230+
- Just Dire Things 1.5.7+
