# JDT Extras

JDT Extras (`jdte`) 是 [Just Dire Things](https://www.curseforge.com/minecraft/mc-mods/just-dire-things) 的 NeoForge 扩展模组，围绕 JDT 机器增加升级卡、更多升级槽、时间加速器和若干自动化机器。

当前版本：`0.5.0`

[English README](README_EN.md)

## 功能

### 升级卡（11 种）

升级卡可以安装到支持的 JDT/JDTE 机器中。普通机器有 4 个升级槽，扩展机器有 8 个升级槽。

| 升级卡 | 效果 | 每台机器上限 |
|---|---|---|
| 容量升级 | 机器 FE 容量和流体容量翻倍 | 3 |
| 超频升级 | 强制 1 tick 运行并每 tick 执行两次，耗电变为 3 倍 | 1 |
| 降频升级 | 强制 40 tick 运行，耗电降低 80% | 1 |
| 流体升级 | 仅使流体容量翻倍 | 3 |
| 存储流体升级 | 仅限点击器，添加内部流体储罐并自动填充槽位内的流体容器 | 1 |
| 发电机升级 | 仅限发电机，消耗 2 倍燃料输入，获得 3 倍发电量 | 1 |
| 范围升级 | 仅限范围机器，使可调范围上限翻倍 | 2 |
| 过滤升级 | 为可过滤机器添加额外过滤槽，每张增加 9 个槽位 | 2 |
| 创造升级 | 机器无需消耗 FE；时间加速器无需消耗时间流体；自带超频效果 | 1 |
| 抢夺升级 | 生物粉碎机专用，增加额外掉落概率，最高 6 级 | 6 |
| 锋利升级 | 生物粉碎机专用，增加攻击伤害，最高 35 点 | 6 |

超频升级和降频升级不可同时安装。

### 时间加速器

- **初级时间加速器**：仅消耗 JDT 时间流体，默认 4x 加速，安装超频或创造升级后为 16x。
- **高级时间加速器**：消耗 JDT 时间流体和 FE，速度可调 1-128x，安装超频或创造升级后为 256x。
- **扩展高级时间加速器**：高级时间加速器的扩展版本，拥有 8 个升级槽。

**时间流体催化剂** 可以直接触发源水到 JDT 时间流体的 FluidDrop 转换。

时间加速器支持范围配置、红石控制和过滤槽，会加速范围内可被 JDT 时间魔杖加速的方块或方块实体。

### 扩展高级机器

使用 **扩展升级** 右键指定 JDT T2 机器，可以转换为 JDTE 扩展版本并获得 8 个升级槽。

- 扩展高级点击器
- 扩展高级方块破坏器
- 扩展高级方块放置器
- 扩展高级方块替换器
- 扩展高级投掷器
- 扩展高级传感器
- 扩展高级流体收集器
- 扩展高级流体放置器
- 扩展高级时间加速器

### 额外自动化机器

- 粘胶激活器：初级、高级、扩展
- 凝胶发生器：高级、扩展
- 流体稳定器：初级、高级、扩展；使用催化剂槽匹配 JDT FluidDrop 配方，在配置范围内直接转换源流体
- 物品放置器：初级、高级、扩展
- 流体放置器：初级、高级、扩展
- 物品接收器：初级、高级、扩展
- 流体接收器：初级、高级、扩展
- 生物粉碎机：高级、扩展；杀死生物产生掉落物和经验流体，支持抢夺和锋利升级，可放置在刷怪笼上方阻止刷怪

### 生物粉碎机

生物粉碎机是一种新机器，可以杀死范围内的生物并产生掉落物和经验流体。

- **高级生物粉碎机**：4 个标准升级槽 + 2 个专用升级槽
- **扩展生物粉碎机**：8 个标准升级槽 + 2 个专用升级槽

专用升级槽支持：
- **抢夺升级**：最高 6 级，每级增加 50% 额外掉落概率
- **锋利升级**：最高 6 个，每个增加 5 点伤害

特殊功能：放置在刷怪笼上方可阻止刷怪，并直接产生掉落物和经验流体。

### BOSS 精华

生物粉碎机可以粉碎 BOSS 并产生独特精华：
- 凋灵精华
- 末影龙精华
- 远古守卫者精华

## 安装

1. 安装 Minecraft `1.21.1`。
2. 安装 NeoForge `21.1.230+`。
3. 安装 Just Dire Things `1.5.7+`。
4. 将 `jdte-x.x.x.jar` 放入客户端和服务器的 `mods` 文件夹。

## 依赖

- Minecraft `1.21.1`
- NeoForge `21.1.230+`
- Just Dire Things `1.5.7+`
- Java `21`

## 开发构建

本项目通过 Gradle 构建。开发环境需要本地存在 JDT jar：

```text
/home/guili/libs/justdirethings-1.5.7.jar
```

常用命令：

```bash
./gradlew compileJava
./gradlew jar
./gradlew runClient
./gradlew runServer
```

## 项目结构

- `src/main/java/com/jdte/common`：方块实体、方块、容器、物品、升级系统、网络逻辑。
- `src/main/java/com/jdte/client`：客户端界面、渲染和客户端初始化。
- `src/main/java/com/jdte/mixin`：对 JDT 和客户端界面的运行时注入。
- `src/main/resources/assets/jdte`：语言文件、模型、贴图和 GuideME 文档。
- `src/main/resources/data/jdte`：配方、战利品表等数据文件。
